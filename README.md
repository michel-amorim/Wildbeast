# Wildbeast

Aplicando conhecimentos com CSS display Grid

## Sobre o projeto

_Wildbeast_ é uma aplicação Frontend web e mobile contruída juntamente ao curso "CSS Grid Layout", com intuito de treinar habilidades CSS3.

Layout foi desenvolvido pelo Tutor do curso Origamid, André Rafael.
